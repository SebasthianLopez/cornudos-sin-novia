# 🍸 Cornudos sin Novia

El ranking oficial de la joda. PWA para un grupo cerrado de amigos: registrá tragos,
rechazos, MVP de la noche, retos, apuestas con puntos y el feed de fotos/videos de cada
salida — y peleá por el primer puesto del ranking.

## Correr local

```bash
npm install
npm run dev   # http://localhost:5173
```

## Stack

Vite + React 19 + TypeScript + Tailwind CSS + Supabase (Postgres + Realtime + Storage).
Ver `CLAUDE.md` para la arquitectura y `supabase/schema.sql` para el backend.
